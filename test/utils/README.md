> image splitter testing utility 

## Run on Windows

 run the script with the project virtual environment:

```powershell
..\..\.venv\Scripts\python.exe main.py
```

to use `python` directly, activate the virtual environment first:

```powershell
..\..\.venv\Scripts\Activate.ps1
python main.py
```

## Dependency

The script uses Pillow. If the environment does not already have it, install the local requirements file:

```powershell
..\..\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

## Example

The folder includes `stanford.jpg`, which you can use for a quick test run.
